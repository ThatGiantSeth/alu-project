# ALU Project
for ECE 443 Computer Architectures - Fall 2025

by Seth Myers and Aubrey McKinney

This project implements a simple ALU for later use in a MIPS-based processor.

It support several operations: addition, subtraction, multiplication, passthrough A, and passthrough B. Each operation was built on its own, then connected together into a single unit. All operations, with the exception of multiplication, were implemented using the structural VHDL model. Multiplication was implemented using the behavioral model.
